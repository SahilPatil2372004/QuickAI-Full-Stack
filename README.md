1. Visit the official Node.js website
   Download the latest Node.js version

2. Setup Neon, Clerk, Gemini API

3. Open project folder in VS code
   Open "server" folder in integrated terminal
   Install dependencies using command : npm install
   Run server using command : npm run server



NOTE : Before running client make sure server is running

4. Open client folder in integrated terminal
   Intall dependencies using command : npm install
   Now run project using command : npm run dev
